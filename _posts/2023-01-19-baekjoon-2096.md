## Recap

### 접근법

점화식 이용. 처음에는 n-1번째 row와 n-2번째 row를 비교해야 한다고 생각했는데 그럴필요 없었다. 단순히 n-1번째의 모든 경우에 대해 합을 구하고 누적하면 된다.

n번째 row의 경우 최대 점수를 구하는 경우, 첫번째 칸에는 n-1번째 row의 첫번째 칸, 두번째 칸 중 큰 수가 온다. 마찬가지로 두번째 칸에는 모든 칸에서의 가장 큰 수, 세번째 칸에는 두번째, 세번째 칸 중 큰 수가 온다. 이를 이용해서 단순히 비교 후 더해나가면 된다.

### 시간복잡도

O(n)

### 공간복잡도

초기에는 모든 row의 누적 값을 배열에 저장하였는데, 이경우 메모리 초과가 발생한다.

$100000 * 3 * 4 * 3$ : 약 4MB에 근접한다.

## Solution Code

```cpp
#include <iostream>
#include <algorithm>
#include <vector>
#define MAX 100001

using namespace std;

void solve();

int N;
int map[MAX][3];

int maxsum[3];
int minsum[3];

int maxans, minans;

int main() {
    freopen("input.txt", "r", stdin);
    
    cin >> N;
    for (int i = 0; i < N; i++) {
        for (int j = 0; j < 3; j++) {
            cin >> map[i][j];
        }
    }

    solve();

    cout << maxans << " " << minans << "\n";
}

void solve() {
    for (int i = 0; i < 3; i++) {
        maxsum[i] = map[0][i];
        minsum[i] = map[0][i];
    }

    for (int idx = 1; idx < N; idx++) {
        int tempmax0 = max(maxsum[0], maxsum[1]);
        int tempmax1 = max(max(maxsum[0], maxsum[1]), maxsum[2]);
        int tempmax2 = max(maxsum[1], maxsum[2]);

        maxsum[0] = tempmax0 + map[idx][0];
        maxsum[1] =  tempmax1 + map[idx][1];
        maxsum[2] = tempmax2 + map[idx][2];

        int tempmin0 = min(minsum[0], minsum[1]);
        int tempmin1 = min(min(minsum[0], minsum[1]), minsum[2]);
        int tempmin2 = min(minsum[1], minsum[2]);

        minsum[0] = tempmin0 + map[idx][0];
        minsum[1] = tempmin1 + map[idx][1];
        minsum[2] = tempmin2 + map[idx][2];
    }

    maxans = max(max(maxsum[0], maxsum[1]), maxsum[2]);
    minans = min(min(minsum[0], minsum[1]), minsum[2]);
}
```
