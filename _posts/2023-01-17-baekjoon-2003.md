## Recap

### 접근법

부분합을 구하는 문제.

반복문이나 재귀함수로 접근하면 시간제한에 걸린다.

[Sliding Window Algorithm](https://nt.fakecoding.com/algorithm-slidingwindow) 이용해서 접근.

Indexed Tree로도 접근할 수 있는 것으로 보이지만, 단순한 문제라 Sliding window algorithm 사용.

### 시간복잡도

O(N)

### 얻어갈 점

- Sliding Window Algorithm

Two pointer algorithm의 한 테크닉이다.

```cpp
while (true) {
    // 탈출조건

    // if (sum < M)
        // end index를 1 더하고, 새로운 end에 해당하는 값 더하기
    // else if (sum > M)
        // start index를 1 더하고, 기존 start에 해당하는 값 빼기
    // else
        // 경우의수 + 1
}
```

## Solution Code

```cpp
#include <iostream>
#include <vector>

using namespace std;

void solve();

int N, M;
vector<int> v;
int ans;

int main() {
    freopen("input.txt", "r", stdin);

    cin >> N >> M;
    for (int i = 0; i < N; i++) {
        int n;
        cin >> n;
        v.push_back(n);
    }

    solve();

    cout << ans << "\n";
}

void solve() {
    int start, end, sum;
    start = end = 0;
    sum = v[0];

    while (true) {
        if (start == N || end == N) {
            break;
        }

        if (sum < M) {
            end++;
            sum += v[end];
        } else if (sum > M) {
            sum -= v[start];
            start++;
        } else {
            ans++;
            end++;
            sum += v[end];
        }
    }
}
```
