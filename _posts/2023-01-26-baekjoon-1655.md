## Recap

### 접근법

힙을 이용한, 중앙값 구하기 Task

### 시간복잡도

매 입력마다 O(log n)

### 공간복잡도

### Missed Points

### 얻어갈 점

#### 중앙값 구하기

새로 들어오는 데이터가 기존 데이터의 중앙값보다 작으면 최대 힙에 저장. 크면 최소 힙에 저장하는 방식을 통해, 중앙값을 계산할 수 있게 됨.

매 입력마다, 힙의 크기를 맞춰줘야 함. Ex) maxheap 이 큰데, 중앙값보다 작은 값이 들어오면 maxheap의 top을 minheap에 넣어주고 maxheap에 push

## Solution Code

```cpp
#include <iostream>
#include <queue>
#include <vector>

using namespace std;

int N;
priority_queue<int> maxHeap;
priority_queue<int, vector<int>, greater<int> >  minHeap;

int main() {
    freopen("input.txt", "r", stdin);
    ios_base::sync_with_stdio(false);
    cin.tie(NULL); cout.tie(NULL);
    cin >> N;
    int median;
    for (int i = 0; i < N; i++) {
        int n;
        cin >> n;

        if (maxHeap.size() == 0) {
            maxHeap.push(n);
        } else if (maxHeap.size() < minHeap.size()) {
            if (median > n) {
                maxHeap.push(n);
            } else {
                int temp = minHeap.top();
                minHeap.pop();
                maxHeap.push(temp);
                minHeap.push(n);
            }
        } else if (maxHeap.size() > minHeap.size()) {
            if (median < n) {
                minHeap.push(n);
            } else {
                int temp = maxHeap.top();
                maxHeap.pop();
                minHeap.push(temp);
                maxHeap.push(n);
            }
        } else {
            if (median < n) {
                minHeap.push(n);
            } else {
                maxHeap.push(n);
            }
        }

        if (maxHeap.size() == minHeap.size()) {
            if (maxHeap.top() < minHeap.top()) {
                median = maxHeap.top();
            } else {
                median = minHeap.top();
            }
        } else {
            if (maxHeap.size() < minHeap.size()) {
                median = minHeap.top();
            } else {
                median = maxHeap.top();
            }
        }
        cout << median << "\n";
    }
}
```
