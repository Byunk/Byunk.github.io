## Recap

### 접근법

DFS 이용한 완전탐색 (탈출 조건에 유의)

로직 자체는 어렵지 않았지만, 시간복잡도 때문에 cache 로직을 추가로 구현해야 했음.

### 시간복잡도

50 * 50 의 경우의 수가 존재하므로 worst case 에서 약 2500!의 경우의 수 발생

### 공간복잡도

### Missed Points

1. 처음에 DFS만 이용해서 푼 관계로 시간 초과에 걸림.
2. 탈출 조건을 for 문 안에 넣은 것과 밖에 넣은 것에 걸리는 시간이 달랐음.
3. dfs 함수의 cnt 잘 생각하지 못해 실수했음.

### 얻어갈 점

1. DP (Caching)
mapCnt 배열을 따로 정의하여 방문하였을 때 가장 큰 cnt 를 저장함.

DFS 도중 해당 노드를 방문하였을 때, 배열 안의 값보다 cnt 가 작으면 return.

## Solution Code

```cpp
#include <iostream>

#define MAX (51)

using namespace std;

void solve();
void dfs(int x, int y, int cnt);

char map[MAX][MAX];
bool visited[MAX][MAX];
int mapCnt[MAX][MAX];

int dr[4] = {-1, 1, 0, 0};
int dc[4] = {0, 0, -1, 1};

int R, C;
int ans = -1;
bool isInfinite;

int main() {
    freopen("input.txt", "r", stdin);

    cin >> R >> C;
    for (int i = 0; i < R; i++) {
        for (int j = 0; j < C; j++) {
            cin >> map[i][j];
        }
    }

    solve();

    // 만약 형택이가 동전을 무한번 움직일 수 있다면 -1을 출력한다.
    if (ans == -1 || isInfinite) {
        cout << -1 << "\n";
    } else {
        cout << ans << "\n";
    }
}

void solve() {
    // 보드 가장 왼쪽 위에 동전 올려놓기
    int x = 0;
    int y = 0;
    visited[0][0] = true;

    dfs(x, y, 1);
}

void dfs(int x, int y, int cnt) {
    if (isInfinite) {
        return;
    }

    if (cnt <= mapCnt[x][y]) {
        return;
    } else {
        mapCnt[x][y] = cnt;
    }

    int num = map[x][y] - '0';
    for (int i = 0; i < 4; i++) {
        // 고른 방향으로 X만큼 이동
        int nx = x + num * dr[i];
        int ny = y + num * dc[i];

        // 동전이 보드의 바깥으로 나가면 종료
        if (nx < 0 || ny < 0 || nx >= R || ny >= C) {
            // cout << x << y << cnt << "\n";
            if (cnt > ans) {
                ans = cnt;
            }
            continue;
        } else if (map[nx][ny] == 'H') {
        // 동전이 구멍에 빠지면 종료
            if (cnt > ans) {
                ans = cnt;
            }
            continue;
        } else if (visited[nx][ny]) {
            isInfinite = true;
            return;
        }

        visited[nx][ny] = true;
        dfs(nx, ny, cnt + 1);
        visited[nx][ny] = false;
    }
}
```
