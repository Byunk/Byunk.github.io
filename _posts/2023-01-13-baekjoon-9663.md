## Recap

### 접근법

2차원 배열의 문제를 1차원으로 simplify.

for문의 개수를 줄일 수 있는 접근방법 + dfs.

### 시간복잡도

Greedy로는 N^N으로 시간초과 발생함.

풀이에서는 N^2으로 단순화.

### 공간복잡도

### Missed Points

1. 2차원 배열로 구현하였다가 시간초과 발생.

2. `#define ABS(X) ((X>0) ? (X) : (-X))` 은 parameter로 arithmetic 연산이 못들어감. 여기서 오류 발생.

=> cstdlib의 abs 사용

3. 탈출조건에서 2중 for 사용하여 시간초과 발생.

### 얻어갈 점

1. 문제 단순화 기법

2차원 배열을 각 열에 queen이 반드시 1개만 있어야 함에서 착안하여 1차원 배열 문제로 단순화

2. for문 단순화.

for문을 2개만 사용하여, 모든 조건 탐색.

```cpp

void dfs(int r) {
    // r: row
    if (r == N) {
        ans++;
        return;
    }
    
    // 열
    for (int j = 0; j < N; j++) {
        // r, j 에 대해서 테스트
        bool flag = true;
        for (int i = 0; i < r; i++) {
            if (xLocation[i] == j || xLocation[i] + (r - i) == j || xLocation[i] - (r - i) == j) {
                xLocation[r] = j;
                flag = false;
                break;
            }
        }
        if (flag) {
            xLocation[r] = j;
            dfs(r+1);
        }
    }
}
```

## Solution Code

```cpp
#include <iostream>
#include <cstdlib>
#define MAX (16)

using namespace std;

void solve();
void dfs(int r);

int N;
int xLocation[MAX];
int ans;

int main() {
    freopen("input.txt", "r", stdin);

    cin >> N;
    solve();
    cout << ans << "\n";
}

void solve() {
    dfs(0);
}

void dfs(int r) {
    // r: row
    if (r == N) {
        ans++;
        return;
    }
    
    // 열

        // r, j 에 대해서 테스트
        bool flag = true;
        for (int i = 0; i < r; i++) {
            if (xLocation[i] == j || xLocation[i] + (r - i) == j || xLocation[i] - (r - i) == j) {
                xLocation[r] = j;
                flag = false;
                break;
            }
        }
        if (flag) {
            xLocation[r] = j;

        }
    }
}
```
