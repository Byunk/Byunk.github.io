## Recap

### 접근법

Recursive로 접근하면 시간제한에 걸린다. 2^90 > 10^9

[피보나치 알고리즘](https://shoark7.github.io/programming/algorithm/%ED%94%BC%EB%B3%B4%EB%82%98%EC%B9%98-%EC%95%8C%EA%B3%A0%EB%A6%AC%EC%A6%98%EC%9D%84-%ED%95%B4%EA%B2%B0%ED%95%98%EB%8A%94-5%EA%B0%80%EC%A7%80-%EB%B0%A9%EB%B2%95)

위 링크 참고. 단순하게 O(n)을 이용해서 해결.

### 시간복잡도

O(n)

### 공간복잡도

### Missed Points

- 피보나치 수의 n이 90에 이르면, int type에서 overflow 발생함.

=> long long으로 변경

### 얻어갈 점

- 피보나치 (반복적 풀이)

```cpp
void solve() {
    long long sum = 0;
    long long a = 0;
    long long b = 1;
    for (int i = 2; i <= N; i++) {
        long long temp = a + b;
        a = b;
        b = temp;
    }
    ans = b;
}
```

#include <iostream>
