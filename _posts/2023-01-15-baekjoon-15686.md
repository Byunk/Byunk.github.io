## Recap

### 접근법

시간복잡도 추산에 오류가 있어, 접근방법을 찾는데 시간이 좀 걸렸다.

DFS를 이용한 완전탐색

### 시간복잡도

DFS : O(2^N)

### 공간복잡도

### Missed Points

- 초기에 STL 사용을 너무 어렵게해서 접근이 어려웠음.

- DFS에서 치킨집을 제거할 때마다 distance를 계산하면 시간초과에 걸릴줄 알았는데 걸리지 않음.

=> 단순한 방법으로 먼저 접근하고, 디벨롭

### 얻어갈 점

- `abs()`

`abs()`를 사용하려면 `#include <stdlib>`

- 벡터나 array의 인덱스를 이용해서 overhead 줄이기.

## Solution Code

```cpp
#include <iostream>
#include <vector>
#include <cstdlib>
#define MAX 51

using namespace std;

void solve();
void dfs(int idx, int remain);
int distance(int idxHouse);

int N, M;
int map[MAX][MAX];
vector<pair<pair<int, int>, int> > store;
vector<pair<int, int> > house;
int ans = 100000;

int main() {
    freopen("input.txt", "r", stdin);

    cin >> N >> M;
    for (int i = 0; i < N; i++) {
        for (int j = 0; j < N; j++) {
            int n;
            cin >> n;
            if (n == 2) {
                store.push_back(make_pair(make_pair(i, j), 1));
            } else if (n == 1) {
                house.push_back(make_pair(i, j));
            }
            map[i][j] = n;
        }
    }
    solve();
    cout << ans << "\n";
}

void solve() {
    dfs(0, store.size()-M);
}

void dfs(int idx, int remain) {
    if (remain == 0) {
        int dist = 0;
        for (int i = 0; i < house.size(); i++) {
            dist += distance(i);
        }
        if (dist < ans) {
            ans = dist;
        }
        return;
    }
    if (idx == store.size()) return;
    map[store[idx].first.first][store[idx].first.second] = 0;
    store[idx].second = 0;
    dfs(idx+1, remain-1);
    map[store[idx].first.first][store[idx].first.second] = 1;
    store[idx].second = 1;

    dfs(idx+1, remain);
}

int distance(int idxHouse) {
    int min = 1000;

    int x = house[idxHouse].first;
    int y = house[idxHouse].second;
    for (int j = 0; j < store.size(); j++) {
        if (store[j].second == 0) continue;
        int dist = abs(store[j].first.first - x) + abs(store[j].first.second - y);
        if (dist < min) {
            min = dist;
        }
    }
    return min;
}
```
