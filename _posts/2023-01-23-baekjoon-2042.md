## Recap

### 접근법

Indexed Tree 이용.

### 시간복잡도

Indexed Tree의 각 연산마다 O(log n) (update, 구간합)

=> (M+K) * (log N)

### 공간복잡도

### Missed Points

input에 대한 범위에서 실수가 하나 있어서 런타임 에러가 발생 (int -> long long)

### 얻어갈 점

#### Indexed Tree 범위

Indexed Tree를 위한 배열을 선언할 때, MAX * 2 만큼의 크기로 선언을 하였는데, MAX * 4정도의 크기로 선언을 해야했음. 그 이유는 원소의 개수가 2^N + 1 개만큼 존재할 때 트리의 크기가 2 * 2^(N+1) 까지 늘어날 수 있기 때문임. (2^N - 1만큼의 빈공간 때문).

*따라서 앞으로는 Indexed Tree의 구현에서 최소 4 * N만큼의 크기를 설정할 계획.*

```cpp
#include <iostream>
#include <cmath>
#include <vector>
#define MAX 1000001

using namespace std;

void solve();

// M은 수의 변경이 일어나는 횟수이고, K는 구간의 합을 구하는 횟수이다.
int N, M, K;
long long num[MAX];
long long tree[4000000];
vector<long long> out;

int main() {
    freopen("input.txt", "r", stdin);

    cin >> N >> M >> K;
    for (int i = 0; i < N; i++) {
        cin >> num[i];
    }

    solve();

    for (int i = 0; i < out.size(); i++) {
        cout << out[i] << "\n";
    }
}

void solve() {
    // Indexed Tree의 크기 구하기
    int depth = 0;
    while (pow(2, depth) < N) {
        depth++;
    }

    // Indexed Tree 생성
    for (int i = 0; i < N; i++) {
        int idx = pow(2, depth) + i;
        tree[idx] = num[i];

        while (idx / 2 > 0) {
            idx = idx / 2;
            tree[idx] = tree[2*idx] + tree[2*idx + 1];
        }
    }

    for (int i = 0; i < M + K; i++) {
        int a;
        long long b, c;
        cin >> a >> b >> c;
        if (a == 1) {
            int idx = pow(2, depth) + b - 1;
            tree[idx] = c;

            while (idx/2 > 0) {
                idx = idx / 2;
                tree[idx] = tree[2*idx] + tree[2*idx + 1];
            }
        } else if (a == 2) {
            int left = pow(2, depth) + b - 1;
            int right = pow(2, depth) + c - 1;
            long long sum = 0;

            while (left <= right) {
                if (left % 2 == 0) {
                    left = left / 2;
                } else {
                    sum += tree[left];
                    left = left / 2 + 1;
                }

                if (right % 2 == 0) {
                    sum += tree[right];
                    right = right / 2 - 1;
                } else {
                    right = right / 2;
                }
            }
            out.push_back(sum);
        }
    }
}
```
