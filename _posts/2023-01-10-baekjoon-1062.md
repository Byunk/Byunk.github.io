## Recap

### 접근법

DFS, 완전탐색

알파벳 조합을 묻는 문제이므로, DFS로 접근

### 시간복잡도

$_{26}C_{13}$

### 공간복잡도

### Missed Points


### 얻어갈 점

1. bool 배열에 char 저장하는 전략

```cpp
alphabet['a' - 'a'] = true;
```

2. Alphabet index로 DFS

```cpp
void dfs(int idx, int cnt)
```

## Solution Code

```cpp
#include <iostream>
#include <string>
#include <vector>

using namespace std;

void solve();
void dfs(int idx, int cnt);
int numValid();

int N, K;
vector<string> words;
bool alphabet[26];

int ans = -1;

int main() {
    // freopen("input.txt", "r", stdin);

    cin >> N >> K;
    for (int i = 0; i < N; i++) {
        string word;
        cin >> word;
        words.push_back(word);
    }

    solve();
    cout << ans << "\n";
}

void solve() {
    if (K < 5) {
        ans = 0;
        return;
    }

    alphabet['a' - 'a'] = true;
    alphabet['n' - 'a'] = true;
    alphabet['t' - 'a'] = true;
    alphabet['i' - 'a'] = true;
    alphabet['c' - 'a'] = true;

    dfs(0, K - 5);
}

void dfs(int idx, int cnt) {
    if (cnt == 0) {
        int result = numValid();
        if (ans < result) {
            ans = result;
        }
        return;
    }

    if (idx >= 26) {
        return;
    }

    if (alphabet[idx] == false) {
        alphabet[idx] = true;
        dfs(idx + 1, cnt - 1);
        alphabet[idx] = false;
    }

    dfs(idx + 1, cnt);
}

int numValid() {
    int cnt = 0;

    for (int i = 0; i < N; i++) {
        string word = words[i];
        bool flag = true;
        for (int j = 0; j < word.size(); j++) {
            if (alphabet[word[j] - 'a'] == false) {
                flag = false;
                break;
            }
        }
        if (flag == true) {
            cnt++;
        }
    }
    return cnt;
}
```
